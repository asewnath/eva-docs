eva.utilities.utils.replace_vars_dict(d, **defs)[source]¶

At the highest level of the dictionary are definitions, such as swell_dir: /path/to/swell. Elsewhere in the dictionary is use of these definitions, such as key: $(swell_dir)/some/file.ext In this script variables like $(swell_dir) are replaced everywhere in the dictionary using the definition.

ddictionary, required

Dictionary to be modified

defs: dictionary, required

Dictionary of definitions for resolving variables expressed as key-word arguments.

d_interp: dictionary

Dictionary with any definitions resolved

eva.utilities.utils.replace_vars_str(s, **defs)[source]¶

Interpolate/replace variables in string

Resolved variable formats is: ${var}. Undefined variables remain unchanged in the returned string. This method will recursively resolve variables of variables.

sstring, required

Input string containing variables to be resolved.

defs: dict, required

dictionary of definitions for resolving variables expressed as key-word arguments.

s_interp: string

Interpolated string. Undefined variables are left unchanged.
